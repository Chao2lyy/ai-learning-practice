
# AI Learning Practice

AI 实践项目


## 环境要求

- Python 3.11+
- PyTorch 2.0+
- CUDA 11.8+（如需 GPU 加速）


## 项目结构

```
ai-learning-practice/
├── README.md              # 项目总览（本文件）
├── mnist_classifier/      # 第一阶段：MNIST 手写数字分类器
│   ├── README.md
│   ├── main.py
│   ├── requirements.txt
│   ├── data/
│   └── ...
└── minGPT/               # 第二阶段：minGPT 实现
    ├── README.md
    ├── model.py
    ├── dataset.py
    ├── trainer.py
    ├── requirements.txt
    ├── data/
    └── ...
```

## 快速开始

### 克隆仓库
```bash
git clone https://github.com/Chao2lyy/ai-learning-practice.git
cd ai-learning-practice
```
