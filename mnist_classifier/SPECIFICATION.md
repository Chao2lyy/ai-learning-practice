
# 手写数字分类器项目规格说明

## 1. 项目概述
本项目使用 PyTorch 实现 MNIST 手写数字分类器，帮助零基础开发者学习深度学习基础概念。

## 2. 学习目标
- 理解深度学习的基本流程
- 掌握 PyTorch 核心概念：Tensor、Dataset、DataLoader、Model、Loss、Optimizer
- 学会搭建简单的全连接神经网络
- 理解训练循环的完整过程
- 能够评估模型性能

## 3. 技术栈
- Python 3.x
- PyTorch (深度学习框架)
- torchvision (计算机视觉工具库)
- matplotlib (可视化)

## 4. 功能需求

### 4.1 数据集处理
- 自动下载 MNIST 数据集
- 数据预处理：归一化、张量转换
- 划分训练集和测试集

### 4.2 模型架构
- 3 层全连接网络
  - 输入层：784 个神经元 (28x28 像素展平)
  - 隐藏层 1：128 个神经元，ReLU 激活
  - 隐藏层 2：64 个神经元，ReLU 激活
  - 输出层：10 个神经元 (数字 0-9)

### 4.3 训练功能
- 使用交叉熵损失函数
- 使用 Adam 优化器
- 完整的训练循环
- 实时显示训练进度和损失
- 记录训练过程

### 4.4 评估功能
- 在测试集上评估模型准确率
- 目标准确率：≥ 95%

## 5. 核心概念讲解（会在代码中详细注释）

### 5.1 Tensor（张量）
- PyTorch 的核心数据结构
- 类似 NumPy 数组，但可以在 GPU 上运行

### 5.2 Dataset（数据集）
- 抽象类，表示数据集
- 负责加载和返回单个样本

### 5.3 DataLoader（数据加载器）
- 批量加载数据
- 支持 shuffle、并行加载等功能

### 5.4 Model（模型）
- 继承自 nn.Module
- 定义网络结构和前向传播

### 5.5 Loss Function（损失函数）
- 衡量预测与真实值的差距
- 本项目使用 CrossEntropyLoss

### 5.6 Optimizer（优化器）
- 更新模型参数以最小化损失
- 本项目使用 Adam

### 5.7 训练循环
- 前向传播 → 计算损失 → 反向传播 → 更新参数

## 6. 验收标准
- 代码完整可运行
- 模型在测试集上准确率 ≥ 95%
- 代码中有详细的中文注释解释每个概念
- 包含训练过程可视化
